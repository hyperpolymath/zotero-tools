Zotero reference management extensions and integrations.

== Components

* `zoterho/` - Core Zotero integration
* `zoterho-template/` - Zotero template system
* `formbd/` - Zotero form-based data entry
* `librarian/` - Automated library management
* `nesy/` - Neurosymbolic Zotero integration
* `rescript-templater/` - ReScript-based template engine
* `safe-storage/` - Secure Zotero storage
* `voyant-export/` - Voyant Tools export integration
* `zotpress/` - Zotero WordPress integration
